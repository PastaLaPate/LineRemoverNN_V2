# Line Remover NN
### A model trained to remove the lines from an scanned paper image.
## Installation

> [!NOTE]  
> Using UV is recommended and the doc will use it.
>

The project will try to install torch with CUDA 13.0 by default.

1. Clone the repo
`git clone https://github.com/PastaLaPate/LineRemoverNN_V2.git`

2. Install dependencies
`uv sync`

3. Run the cli
`uv run lineremovernn`
You should be able to see if cuda is available.
